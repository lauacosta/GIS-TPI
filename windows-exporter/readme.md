# Carga automática de capas a postgis

En la buscador de Windows busca **OSGeo4W Shell.** Se debería haber instalado por defecto cuando instalaste QGIS.

Desde esa Shell movete a donde hayas descomprimido el zip [sig_ign](https://drive.google.com/file/d/12ttDt5NZdEb4_EbFp33CLW6r9PmxhJOY/view?usp=drive_link) que esta en el drive. Entra a la carpeta sig_ign y copia el archivo subir_todo.bat y correlo.
